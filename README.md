# 🍔 Bergmann Restaurante - Landing Page Profissional

Uma landing page moderna, rápida e focada em conversão para restaurantes locais.

> Projeto desenvolvido com foco em atrair clientes, aumentar pedidos e melhorar a presença digital de negócios físicos.

---

## 🚀 Sobre o Projeto

O **Bergmann Restaurante** é uma página web desenvolvida para representar digitalmente um restaurante/lancheria, com foco total em:

- 📲 Conversão de visitantes em clientes
- ⚡ Carregamento rápido
- 🎯 Experiência simples e direta
- 💬 Integração com WhatsApp

Este projeto foi pensado especialmente para **negócios locais que não possuem presença digital forte**, criando uma solução acessível e altamente eficiente.

---

## 🧠 Estratégia por trás do projeto

Diferente de sites comuns, essa landing foi construída com base em princípios de marketing:

- Hero section chamativa (impacto imediato)
- Botões de ação (CTA) bem posicionados
- Design focado em mobile
- Estrutura pensada para decisão rápida
- Redução de distrações
